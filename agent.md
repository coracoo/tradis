# Agent 行为约束

## 逻辑循环中止规则

- 定义：当我连续 3 次在同一问题点上重复“搜索/改动/验证”而没有产生新的可验证进展（例如：反复改同一处但结果不变、反复报同一个错误且无新增线索），视为陷入逻辑循环。
- 计数：每次出现上述“无新增进展”的回合，循环计数 +1；一旦出现新的可验证进展（例如：找到明确根因、拿到新的关键日志/证据、修复并通过验证），循环计数清零。
- 触发：循环计数达到 3 次时，我必须立刻中止当前任务，不再继续尝试“同一方向”的修复。
- 中止输出：中止时必须输出
  - 已完成的开发进度（已改文件与要点）
  - 当前阻塞点与循环原因（为什么会重复）
  - 可选的 2-3 个下一步决策选项（由你拍板）
